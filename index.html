// ── VUE MOIS PAR MOIS + RUNWAY ────────────────────────────── const monthly = useMemo(() => { const todayStr = new Date().toISOString().split('T')[0]; const rows = {}; const sorted = [...transactions].sort((a,b) => a.date < b.date ? -1 : 1); sorted.forEach(t => { if (isFraisPersoActif(t)) return; const k = t.date.slice(0,7); if (!rows[k]) rows[k] = { key:k, inNet:0, out:0, salaire:0, nbEch:0, futur:false }; const tax = calculateLineTax(t); const isSal = /paie|salaire/i.test(t.title || ''); if (t.type === 'income' || t.type === 'adjustment') { rows[k].inNet += tax.net; if (t.groupId) rows[k].nbEch++; } else { if (isSal) rows[k].salaire += parseFloat(t.amount||0); else rows[k].out += parseFloat(t.amount||0); } if (!t.validated) rows[k].futur = true; }); const keys = Object.keys(rows).sort(); let run = 0; const list = keys.map(k => { const r = rows[k]; run += r.inNet - r.out - r.salaire; return { ...r, solde: run, isPast: k < todayStr.slice(0,7), isNow: k === todayStr.slice(0,7) }; }); // runway : nb de mois pleins avant le 1er solde négatif const idxNow = list.findIndex(r => r.isNow); let mois = 0, rupture = null; for (let i = Math.max(idxNow,0); i < list.length; i++) { if (list[i].solde < 0) { rupture = list[i]; break; } mois++; } return { list: list.filter(r => !r.isPast || r.isNow), rupture, mois }; }, [transactions]);
